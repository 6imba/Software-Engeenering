# archive file: a file that is composed of more than one file.
	- file that is created compressing a folder that contains more than one file.

- In computing, an archive file is a computer file that is composed of one or more files along with metadata.
- Archive files are used to collect multiple data files together into a single file for easier portability and storage, or simply to compress files to use less storage space.
- archive files are used for packaging software_files for distribution, as software content are naturally spread across seperate different files.
- So archive is known as package.
